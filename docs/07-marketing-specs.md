# 07 — Spécifications des formats marketing (2025-2026)

Specs à coder comme **presets** dans le moteur de génération (Module 4). Dimensions en
px. Limites de caractères = limites techniques ; le « sweet spot » d'affichage est
signalé. **Zone critique** : sur FB/IG/LinkedIn, seuls ~110-125 premiers caractères sont
visibles avant « …plus » → traiter ces caractères comme le *hook*.

## 1. Brochure (imprimé long format)
| Élément | Spéc |
|---|---|
| Letter | 8,5 × 11 po → **2550 × 3300 px @ 300 ppp** |
| A4 | 210 × 297 mm → **2480 × 3508 px @ 300 ppp** |
| Résolution | 300 ppp, photos haute résolution |
| Fond perdu | +3 mm (~+36 px) par bord |
| Pliage courant | tri-fold (6 panneaux), bi-fold (4), gate-fold (luxe) |
| Format | fiche 1 page recto-verso (visite libre) **ou** 4-8 pages (haut de gamme) |
| Polices | 2-3 max ; palette 2-3 couleurs |

Sections : couverture pleine page + adresse + **prix** · caractéristiques clés
(chambres/sdb/superficie) · description narrative (lifestyle) · galerie photos · plans
d'étage cotés · carte de quartier + commodités · bloc contact (courtier, tél, courriel,
agence, logo) · CTA + code QR.

## 2. Annonce classée moyenne (Kijiji / classées)
| Élément | Spéc |
|---|---|
| Titre | ~70 car. ; mots-clés (type + quartier + accroche) en premier |
| Corps | 300-800 mots structurés |
| Photos | jusqu'à plusieurs ; **1280 × 960 px (4:3)** |
| Structure | accroche → puces bénéfices → détails (prix/superficie/ch/sdb) → CTA + coordonnées |
| QC | Charte de la langue française (FR requis) |

## 3. Fil Facebook
| Élément | Spéc |
|---|---|
| Carré 1:1 | **1080 × 1080** |
| Portrait 4:5 (recommandé mobile) | **1080 × 1350** |
| Aperçu de lien 1,91:1 | **1200 × 630** |
| Texte sur image | < 20 % |
| Texte principal | tech. ~4096 ; **~125 car. visibles** (hook) |
| Titre lien | ~27-40 car. |

## 4. Facebook Marketplace (immobilier)
| Élément | Spéc |
|---|---|
| Photos | jusqu'à **10** (8-10 = ~3× plus de demandes) |
| Format | carré **1080 × 1080** ou 4:5 **1080 × 1350** |
| Titre | accroche courte, meilleur atout |
| Description | ch/sdb/superficie/prix/conditions/commodités ; **pas d'URL** |
| Champs | prix, superficie, nb chambres, type, localisation |

## 5. Instagram
| Élément | Spéc |
|---|---|
| Carré 1:1 | **1080 × 1080** |
| Portrait 4:5 (recommandé) | **1080 × 1350** |
| Paysage | 1080 × 566 |
| Carrousel | **1080 × 1350**, même ratio sur toutes les diapos, 2-20 (optimal 8-10) |
| Légende | 2200 car. max ; ~125 visibles |
| Hashtags | 30 max ; **recommandé 3-5** (jusqu'à 10-15) |

Carrousel : diapo 1 = hook fort ; dernière = CTA.

## 6. X / Twitter (compte standard, non-Premium)
| Élément | Spéc |
|---|---|
| Limite | **280 car.** par post (chaque post d'un fil) |
| Image horizontale | **1600 × 900 (16:9)** |
| Carrée / verticale | 1080 × 1080 / 1080 × 1350 |
| Fils | posts liés, numérotés (1/, 2/) ; un fil = une « visite » |

## 7. LinkedIn
| Élément | Spéc |
|---|---|
| Image lien 1,91:1 | **1200 × 627** |
| Carré 1:1 | 1080 × 1080 |
| Portrait 4:5 | 1080 × 1350 |
| Texte | 3000 car. max ; ~140 (desktop)/~110 (mobile) visibles ; **optimal 150-300** |
| Hashtags | 3-5 en fin |
| Ton | professionnel, expertise/marché |

## 8. Diapos vidéo (carrousel animé)
| Format | Dimensions | Ratio | Usage |
|---|---|---|---|
| Carré | **1080 × 1080** | 1:1 | feed FB/IG |
| Vertical | **1080 × 1920** | 9:16 | Reels/Stories/TikTok |
| Portrait | 1080 × 1350 | 4:5 | feed mobile max |

Nombre : 5-10 diapos (~3-5 s/diapo) ; durée 15-60 s (jusqu'à 90 s Reels) ; 1 idée/diapo,
5-10 mots, texte dans la zone sûre centrale (~250 px de marge bas) ; diapo 1 = hook,
diapo finale = CTA + coordonnées.

## Copy à fort taux de conversion
- **Hook** (1re ligne, ~125 car.) : question / rareté / lifestyle / insight local — pas
  « 3 ch, 2 sdb ».
- **CTA** unique et explicite (« DM "VISITE" », « Réservez votre visite »).
- **Emojis** comme balises (✨ 📍 🗓 🗝 📲) **avec parcimonie** — l'UI Softimmo n'en
  utilise pas, mais le contenu social peut en contenir : rendre **configurable/désactivable**.
- Ratio ~1 promo : 3 valeur (règle 80/20). A/B tester hooks/CTA/longueur.

## Loi 96 (rappel)
Génération **bilingue par défaut** ; FR **prééminent** (≥ autre langue) ; pour le
contenu social, prévoir version FR autonome ou bilingue de sens/valeur équivalents.
